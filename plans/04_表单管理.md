# 表单管理

## 当前状态

### 已完成
- 已完成表单管理页主流程：目录树、本地搜索、分组/表单创建、重命名、移动、删除、顶部详情展示与预览空态已闭环，请求层与 `form-group / form-definition` API 也已收口。
- 已完成左侧结构收口：目录树切到 `el-tree`、滚动容器切到 `el-scrollbar`，创建/重命名/移动/删除类弹窗统一到表单校验实现。
