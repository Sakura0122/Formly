# 表单管理

## 当前状态

### 已完成
- 已完成表单管理页主流程，覆盖目录树、本地搜索、分组/表单创建、重命名、移动、删除、顶部详情展示与预览空态。
- 已完成首页路由调整，请求层复用，以及 `form-group / form-definition` API 模块封装。
- 已完成左侧目录树切到 `el-tree`、滚动容器切到 `el-scrollbar`，并收口创建/重命名/移动/删除类弹窗实现。
- 已完成管理页右侧发布态预览首版接入，并保持“无发布版本时展示空状态”的交互约束。
- 已通过 `npm run type-check` 与 `npm run build` 验证前端构建链路。
- 已完成右侧真实预览：仅在存在已发布版本时展示只读表单，否则固定显示空状态。
- 已完成管理页读取接口收口：右侧预览只消费发布态单个 `schema`，不再接收草稿相关字段。
- 已完成管理页版本展示收口：顶部版本改为展示发布版本号文案 `V1 / V2`，不再直接显示版本 ID。
- 已完成管理页顶部更多操作首版接入：在“数据管理”左侧新增三点下拉，并支持发布态表单导出 PDF。
- 已完成发布态打印导出链路：离屏白底画布经 `snapdom + jspdf` 生成 A4 自适应分页 PDF，文本框/数字在打印态下不再回退 `/` 占位。
- 已通过新增导出能力相关的 `npm run type-check` 与 `npm run build` 验证。
- 已完成 PDF 预览交互收口：导出改为先打开新窗口，再展示生成后的 PDF，而不是直接下载。
- 已完成 PDF 纸张策略收口：取消当前横向自动放大，统一按竖版 A4 分页生成，避免内容发胖、变短与提前换行。
- 已完成更多操作入口样式收口：三点改为无边框纯图标触发器，不再显示按钮边框。
- 已完成 PDF 打开时机收口：先生成 PDF blob，再打开新窗口加载预览，避免先看到空白页。
- 已完成更多操作触发器焦点态收口：清掉三点图标聚焦时的蓝色描边，只保留纯图标视觉。
- 已完成三点触发器 loading 写法收口：改为模板内直接分支渲染，不再使用数组式 `:class`。
- 已完成导出 loading 语义收口：三点触发器恢复为单一纯图标结构，loading 改挂到打印态 `FormPreviewCanvas` 容器。
- 已完成离屏导出容器收口：移除无效的 `v-loading`，不再在不可见画布上表达无效 loading。
- 已完成 PDF 预览页标题收口：改用 `about:blank + iframe` 承载 blob，并手动设置窗口标题为表单文件名，避免标签页显示 UUID。
- 已完成 PDF 文档元数据标题收口：导出时写入 PDF `title/subject` 等元数据，供浏览器内置 PDF 查看器读取显示。
- 已完成管理页 PDF 导出逻辑抽离：导出实现已从 `index.vue` 拆出到独立组合函数，页面只保留触发入口与离屏容器。
- 已完成页面内字符串 helper 收口：移除 `buildPdfFileName / escapeHtml` 等页面内多余函数，文件名与标题拼接内聚到导出实现内部。
- 已完成导出调用方式收口：移除 `useFormManageExportPdf` 的函数式取值参数，改为调用时直接传 `element/formName`。
